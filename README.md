# Gather data from users using process adapted from CyberSignature.

The original paper is - 
Nnamoko, Nonso, et al. "CyberSignature: A user authentication tool based on behavioural biometrics." Software Impacts, vol. 14, 2022, p. 100443.
available [here](https://research.edgehill.ac.uk/ws/portalfiles/portal/58953568/1_s2.0_S2665963822001270_main.pdf)
[Original code](https://github.com/CyberSignature-EHU/CyberSignature)
